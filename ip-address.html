What is an IP Address?

An IP (Internet Protocol) address is a unique string of numbers that identifies a specific device on a network. Just like physical home address, it ensures that data sent over the internet reaches the correct computer or smartphone.

There are 2 main types of IP addresses: IPv4, which looks like 192.168.1.1, and the newer IPv6, which was created to provide more addresses for the billion of devices now online. Each device needs one to communicate globally.

What is Web 2.0?

Web 2.0 is the "social" or "participatory" web that we experience today. It shifted the focus from static pages to user-generated content, and interoperability for end users through pplatforms like social media and blogs.

This version of the web is defined by two-way communication, allowing users to interact with each other, share media and participate in the creationof the internet's content rather than just observing it.

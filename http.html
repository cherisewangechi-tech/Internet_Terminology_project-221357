What is HTTP?

HTTP, which stands for HyperText Transfer Protocol, is the foundation of data exchange on the World Wide Web. It is the protocol used by web browsers to request and receive web pages and images from a web server.

An HTTP transaction follows a structured request-response cycle that begins when a client, such as a web browser, establishes a connection to a server and sends a Request Message containing a method (like GET) and a target URL. The server processes this request and returns a Response Message, which includes a status code (such as 200 OK) and the requested content, after which the connection is typically closed or reused for further requests.

What is the World Wide Web?

The World Wide Web is an information system where documents and other web resources are identifies by URLs, interlinked by hypertext links, and accessible via the Internet.

While people often use the terms interchangeably, the Web is actually a service that runs on top of the Internet infrastructure. It functions as a massive network of digital content—such as HTML documents, images, and videos—that are transferred using the Hypertext Transfer Protocol (HTTP). This system allows users to navigate seamlessly from one resource to another using hyperlinks, creating a global web of information that is indexed by search engines and rendered by web browsers to be readable on a user's device.

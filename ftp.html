What is File Transfer Protocol?

FTP is a standard network protocol used specifically for transferring files between a client anad a server on a computer network.

There are 2 types of FTP: active and passive. Active FTP involves teh server initiating a connecction to a port provided by the client, which often results in connection failures due to modern firewall restrictions. In contrast, Passive FTP requires the client to intitiate the connection to a port opened by the server, making it the preferred method for modern networks as it easily bypasses client-side firewall blocks.

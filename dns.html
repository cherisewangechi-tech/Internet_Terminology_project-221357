What is DNS?

DNS is decntralized system that translates human-friendly domain names, like google.com, into numerical IP addresses that computers use to identify each other.

Without DNS, one would have to remember a long string of numbers for every website they wanted to visit. When you type a URL into your browser, a DNS resolver looks up the address and points your request to the correct server instantly.
